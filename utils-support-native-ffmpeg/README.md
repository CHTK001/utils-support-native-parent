# utils-support-native-ffmpeg

FFmpeg RTMP Rust native 库

---

## 快速开始

### 1. 添加依赖

```xml
<dependency>
    <groupId>com.chua</groupId>
    <artifactId>utils-support-native-ffmpeg</artifactId>
    <version>${project.version}</version>
</dependency>
```

---

## 配置说明

本模块为零配置模块，引入依赖后即可使用。
