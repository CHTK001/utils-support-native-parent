# utils-support-native-rust-proxy

Rust 代理协议 native 库（HTTP/SOCKS5/RDP/SSH/VNC/FTP）

---

## 快速开始

### 1. 添加依赖

```xml
<dependency>
    <groupId>com.chua</groupId>
    <artifactId>utils-support-native-rust-proxy</artifactId>
    <version>${project.version}</version>
</dependency>
```

---

## 配置说明

本模块为零配置模块，引入依赖后即可使用。
